# ItemChanger.Silksong

This is a work-in-progress port of Hollow Knight's [ItemChanger mod](https://github.com/homothetyhk/HollowKnight.ItemChanger) to Hollow Knight Silksong.

## Dependencies

- [ItemChanger.Core](https://github.com/BadMagic100/ItemChanger.Core)
- [Benchwarp](https://github.com/homothetyhk/Silksong.Benchwarp)
